Допълнителни концепции за работа с цели числа

1. Цикъл for

Синтаксис:

for (инициализация; условие; стъпка)
    оператор

Начин на изпълнение:

  1. Изпълнява се инициализацията (обикновено задаване на начална стойност на брояч)
  2. Проверява се условието - ако е истина, изпълнява се операторът
  3. Изпълнява се стъпката (обикновено увеличаване/намаляване на брояча)
  4. Връща се на стъпка 2

Пример:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int i;            // декларация на променлива (виж [Променливи](../02_Целочислени%20типове%20данни.md#2-променливи))
    
    // Отпечатва числата от 1 до 5
    for (i = 1; i <= 5; i = i + 1) {  // for - цикъл с брояч (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for)), (инициализация; условие; стъпка) - кръгли скоби с три части, = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)), <= - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение)), + - събиране (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции))
        printf("%d ", i);              // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип)), ; - край на оператора
    }                                  // } - край на съставния оператор (тяло на цикъла) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    return 0;
}

Изход: 1 2 3 4 5

Терминология:

  • for - оператор за цикъл с брояч. Удобен когато знаем колко пъти искаме да повторим операцията
  • Инициализация - задава начална стойност на променливата-брояч
  • Условие - определя до кога ще продължи цикълът
  • Стъпка - определя как се променя броячът на всяка итерация

Примери за използване:

Въвеждане на n числа:

1.1. Цикъл while

Синтаксис:

while (условие)         // while - цикъл с предусловие, (условие) - кръгли скоби
    оператор            // тяло на цикъла (може да бъде съставен оператор { })

Начин на изпълнение:

  1. Проверява се условието - ако е истина (различно от 0), изпълнява се операторът
  2. След изпълнение на оператора, отново се проверява условието
  3. Цикълът се повтаря, докато условието стане лъжа (равно на 0)

Пример:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int count = 1;            // декларация и инициализация (виж [Променливи](../02_Целочислени%20типове%20данни.md#2-променливи)), = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
    
    // Отпечатва числата от 1 до 5
    while (count <= 5) {      // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), (условие) - кръгли скоби, <= - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("%d ", count); // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип)), ; - край на оператора
        count = count + 1;    // = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)), + - събиране (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)) (увеличаване на брояча)
    }                        // } - край на съставния оператор (тяло на цикъла) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    return 0;
}

Изход: 1 2 3 4 5

Терминология:

  • while - оператор за цикъл с предусловие. Проверява условието преди да изпълни тялото. Ако условието е лъжа от самото начало, тялото няма да се изпълни нито веднъж
  • Условие - израз, който се оценява като истина (ненулева стойност) или лъжа (0)
  • Предусловие - условието се проверява преди изпълнение на тялото

Примери за използване:

Обработка на число докато не стане 0:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int n = 1234;
    int sum = 0;
    int digit;
    
    while (n > 0) {          // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), (условие) - кръгли скоби, > - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        digit = n % 10;      // % - модуло (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
        sum = sum + digit;   // = - присвояване, + - събиране (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции))
        n = n / 10;          // / - деление (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване
    }                        // } - край на съставния оператор (тяло на цикъла) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    printf("Сумата на цифрите: %d\n", sum);  // printf - функция за изход от #include <stdio.h>
    
    return 0;
}

Четене на числа докато не се въведе 0:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int number;
    int sum = 0;
    
    printf("Въведете числа (0 за край): ");
    scanf("%d", &number);    // scanf - функция за вход от #include <stdio.h>, & - оператор за адрес
    
    while (number != 0) {    // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), (условие) - кръгли скоби, != - сравнение за различно (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        sum = sum + number;  // = - присвояване, + - събиране
        scanf("%d", &number); // scanf - функция за вход от #include <stdio.h>, & - оператор за адрес
    }                        // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    printf("Сумата е: %d\n", sum);  // printf - функция за изход от #include <stdio.h>
    
    return 0;
}

1.2. Сравнение между for и while - кога кое да използваме?

Основни разлики:

Характеристика for while
Структура Три части в заглавието: инициализация, условие, стъпка Само условие в заглавието
Инициализация В заглавието на цикъла Преди цикъла
Стъпка (увеличаване/намаляване) В заглавието на цикъла В тялото на цикъла
Кога е подходящ Когато знаем колко пъти искаме да повторим Когато не знаем колко пъти ще се повтори

Кога да използваме for:

for е по-подходящ когато: - Знаем точно колко пъти искаме да повторим операцията - Имаме брояч, който се променя с фиксирана стъпка - Искаме компактност - инициализацията, условието и стъпката са на едно място

Примери: - Отпечатване на числа от 1 до n - Въвеждане на точно n числа - Обработка на всеки елемент в известен диапазон

// Пример: Отпечатване на числа от 1 до 10
for (i = 1; i <= 10; i = i + 1) {  // for - цикъл (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for))
    printf("%d ", i);               // printf - функция за изход от #include <stdio.h>
}                                   // } - край на съставния оператор

Кога да използваме while:

while е по-подходящ когато: - Не знаем колко пъти ще се повтори операцията - Условието зависи от данни, които се променят в тялото - Искаме да проверим условие преди да изпълним тялото - Обработваме данни докато не се достигне определено условие

Примери: - Обработка на цифри на число (докато числото не стане 0) - Четене на данни докато не се въведе специална стойност (напр. 0 или -1) - Повтаряне на операция докато условието е изпълнено

// Пример: Обработка на цифри докато числото не стане 0
while (n > 0) {          // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие))
    digit = n % 10;      // % - модуло
    n = n / 10;          // / - деление
}                        // } - край на съставния оператор

Еквивалентност между for и while:

Всеки for цикъл може да се запише като while цикъл и обратно:

Пример - for като while:

// for цикъл
for (i = 1; i <= 5; i = i + 1) {  // for - цикъл (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for))
    printf("%d ", i);              // printf - функция за изход от #include <stdio.h>
}                                  // } - край на съставния оператор

// Еквивалентен while цикъл
i = 1;                             // = - присвояване (инициализация)
while (i <= 5) {                   // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), <= - сравнение
    printf("%d ", i);              // printf - функция за изход от #include <stdio.h>
    i = i + 1;                     // = - присвояване, + - събиране (стъпка)
}                                  // } - край на съставния оператор

Пример - while като for:

// while цикъл
n = 1234;                          // = - присвояване (инициализация)
while (n > 0) {                    // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), > - сравнение
    digit = n % 10;                // % - модуло, = - присвояване
    printf("%d ", digit);          // printf - функция за изход от #include <stdio.h>
    n = n / 10;                    // / - деление, = - присвояване (стъпка)
}                                  // } - край на съставния оператор

// Еквивалентен for цикъл (по-малко читаем)
for (n = 1234; n > 0; n = n / 10) {  // for - цикъл (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for)), (инициализация; условие; стъпка)
    digit = n % 10;                  // % - модуло, = - присвояване
    printf("%d ", digit);            // printf - функция за изход от #include <stdio.h>
}                                    // } - край на съставния оператор

Практически съвети:

  1. Използвай for когато:
    • Имаш брояч, който се променя с фиксирана стъпка
    • Знаеш колко пъти искаш да повториш операцията
    • Кодът става по-четим с инициализация, условие и стъпка на едно място
  2. Използвай while когато:
    • Условието зависи от данни, които се променят в тялото
    • Не знаеш колко пъти ще се повтори операцията
    • Условието е по-сложно и не се вписва добре в заглавието на for
  3. Читаемост на кода:
    • for е по-компактен и ясен когато имаш брояч
    • while е по-ясен когато условието е по-сложно или зависи от данни в тялото

Резюме:

  • for - за цикли с известен брой итерации и брояч с фиксирана стъпка
  • while - за цикли с неизвестен брой итерации, където условието зависи от данни в тялото

И двата цикъла могат да се използват за едни и същи задачи, но изборът зависи от читаемостта и яснотата на кода.

2. Операции за сравнение

Използват се за сравняване на две стойности. Резултатът е истина (1) или лъжа (0).

Оператор Значение Пример Резултат
== Равно 5 == 5 1 (истина)
== Равно 5 == 3 0 (лъжа)
!= Различно 5 != 3 1 (истина)
> По-голямо 5 > 3 1 (истина)
< По-малко 5 < 3 0 (лъжа)
>= По-голямо или равно 5 >= 5 1 (истина)
<= По-малко или равно 3 <= 5 1 (истина)

Важно: == е за сравнение, = е за присвояване!

Пример:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int a = 5, b = 3;                  // = - оператор за присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
    
    if (a == b) {                      // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, == - оператор за сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("Числата са равни\n");  // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип))
    } else {                           // else - алтернативен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор))
        printf("Числата са различни\n");
    }                                  // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    if (a > b) {                       // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, > - оператор за сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("%d е по-голямо от %d\n", a, b);  // printf - функция за изход от #include <stdio.h>
    }                                  // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    return 0;
}

3. Логически оператори

Използват се за комбиниране на условия.

Оператор Значение Пример Обяснение
&& И (AND) (a > 5) && (a < 10) Истина само ако и двете условия са истина
\|\| ИЛИ (OR) (a < 5) \|\| (a > 10) Истина ако поне едно условие е истина
! НЕ (NOT) !(a == 5) Обръща резултата (истина става лъжа и обратно)

Приоритет: ! > && > \|\|

Пример:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {

4. Работа с цифри на число

Извличане на цифри от число

За да извлечем цифрите на едно число, използваме операциите деление (/) и модуло (%).

Алгоритъм:

  1. Вземаме последната цифра: цифра = число % 10
  2. Премахваме последната цифра: число = число / 10
  3. Повтаряме докато число > 0

Пример - сума на цифрите:

#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int n = 1234;
    int sum = 0;
    int digit;
    
    while (n > 0) {          // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), (условие) - кръгли скоби, > - оператор за сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        digit = n % 10;      // % - операция модуло (остатък) (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
        sum = sum + digit;   // = - присвояване, + - събиране (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции))
        n = n / 10;          // / - деление (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване
    }                        // } - край на съставния оператор (тяло на цикъла) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    printf("Сумата на цифрите е: %d\n", sum);  // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип))
    // Резултат: 1 + 2 + 3 + 4 = 10
    
    return 0;
}

8. Проверка за палиндром

Палиндром - число, което се чете еднакво отляво надясно и отдясно наляво.

Примери: 121, 1221, 12321

Алгоритъм:

  1. Запазваме оригиналното число
  2. Обръщаме числото (извличаме цифрите в обратен ред)
  3. Сравняваме оригиналното с обърнатото
#include <stdio.h>    // препроцесорна директива - включва стандартната библиотека за вход/изход (виж [Препроцесор](../01_Въведение%20в%20С.md#4-препроцесор-и-етапи-на-компилация))

int main(void) {
    int n, original, reversed = 0, digit;  // декларация на променливи (виж [Променливи](../02_Целочислени%20типове%20данни.md#2-променливи)), = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
    
    printf("Въведете число: ");  // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип))
    scanf("%d", &n);            // scanf - функция за вход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип)), & - оператор за адрес
    
    original = n;               // = - оператор за присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)) (запазваме оригиналното число)
    
    // Обръщаме числото
    while (n > 0) {             // while - цикъл с предусловие (виж [Оператор за цикъл с предусловие](../03_Оператори.md#4-оператор-за-цикъл-с-предусловие)), (условие) - кръгли скоби, > - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        digit = n % 10;         // % - модуло (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
        reversed = reversed * 10 + digit;  // = - присвояване, * - умножение (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), + - събиране (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции))
        n = n / 10;             // / - деление (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), = - присвояване
    }                           // } - край на съставния оператор (тяло на while) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    // Проверяваме дали са равни
    if (original == reversed) { // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, == - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("ДА\n");         // printf - функция за изход от #include <stdio.h>
    } else {                    // else - алтернативен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор))
        printf("НЕ\n");          // printf - функция за изход от #include <stdio.h>
    }                           // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    return 0;
}

Обяснение на обръщането:

  • За числото 123:
    • Итерация 1: digit = 3, reversed = 0 * 10 + 3 = 3, n = 12
    • Итерация 2: digit = 2, reversed = 3 * 10 + 2 = 32, n = 1
    • Итерация 3: digit = 1, reversed = 32 * 10 + 1 = 321, n = 0
    • Резултат: reversed = 321

9. Проверка за различни цифри

Проверяваме дали всички цифри на числото са различни (няма повтарящи се).

Алгоритъм:

  1. За всяка цифра проверяваме дали се среща в останалата част на числото
  2. Използваме вложени цикли
#include <stdio.h>

int main(void) {
    int n, temp1, temp2;
    int digit1, digit2;
    int hasDuplicate = 0;       // флаг: 0 = няма дубликати, 1 = има дубликати, = - присвояване
    
    printf("Въведете число: ");
    scanf("%d", &n);            // & - оператор за адрес
    
    temp1 = n;                  // = - присвояване
    
    // Външен цикъл - за всяка цифра
    while (temp1 > 0 && hasDuplicate == 0) {  // while - цикъл, (условие) - кръгли скоби, > - сравнение, && - логически И, == - сравнение
        digit1 = temp1 % 10;    // % - модуло, = - присвояване
        temp1 = temp1 / 10;     // / - деление, = - присвояване
        
        temp2 = temp1;          // = - присвояване (проверяваме в останалата част)
        
        // Вътрешен цикъл - проверяваме дали digit1 се среща още
        while (temp2 > 0) {     // while - вложен цикъл, (условие) - кръгли скоби, > - сравнение
            digit2 = temp2 % 10;  // % - модуло, = - присвояване
            
            if (digit1 == digit2) {  // if - условен оператор, (условие) - кръгли скоби, == - сравнение
                hasDuplicate = 1;   // = - присвояване (намерихме дубликат)
                break;              // break - прекъсване на цикъла
            }                       // } - край на съставния оператор
            
            temp2 = temp2 / 10;     // / - деление, = - присвояване
        }                           // } - край на съставния оператор (тяло на вътрешния while)
    }                               // } - край на съставния оператор (тяло на външния while)
    
    if (hasDuplicate == 0) {       // if - условен оператор, (условие) - кръгли скоби, == - сравнение
        printf("ДА\n");
    } else {                        // else - алтернативен оператор
        printf("НЕ\n");
    }                               // } - край на съставния оператор
    
    return 0;
}

10. Степенуване

Алгоритъм:

    scanf("%d", &a);           // scanf - функция за вход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип)), & - оператор за адрес

    scanf("%d", &n);           // scanf - функция за вход от #include <stdio.h>, & - оператор за адрес
    
    if (n < 0) {               // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, < - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("Отрицателна степен не се поддържа за цели числа\n");  // printf - функция за изход от #include <stdio.h>
    } else if (n == 0) {       // else if - алтернативно условие (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, == - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))

    } else {                   // else - алтернативен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор))
        for (i = 1; i <= n; i = i + 1) {  // for - цикъл (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for)), (инициализация; условие; стъпка) - кръгли скоби, <= - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
            result = result * a;  // = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)), * - умножение (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции))
        }                       // } - край на съставния оператор (тяло на for) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    }                           // } - край на съставния оператор (тяло на if-else if-else) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    printf("%d на степен %d = %d\n", a, n, result);  // printf - функция за изход от #include <stdio.h>
    
    return 0;
}

Пример: 2^3 = 2 * 2 * 2 = 8

    int n, i;
    int isPrime = 1;            // флаг: 1 = просто, 0 = не е просто, = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване))
    
    printf("Въведете число: ");  // printf - функция за изход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип))
    scanf("%d", &n);           // scanf - функция за вход от #include <stdio.h> (виж [Въвеждане и извеждане](../02_Целочислени%20типове%20данни.md#5-въвеждане-и-извеждане-на-данни-от-целочислен-тип)), & - оператор за адрес
    
    if (n < 2) {               // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, < - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        isPrime = 0;           // = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)) (числата по-малки от 2 не са прости)
    } else {                   // else - алтернативен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор))
        for (i = 2; i < n; i = i + 1) {  // for - цикъл (виж [Цикъл for](../03_Оператори.md#6-оператор-за-цикъл-for)), (инициализация; условие; стъпка) - кръгли скоби, < - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
            if (n % i == 0) {   // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, % - модуло (виж [Операции](../02_Целочислени%20типове%20данни.md#3-операции)), == - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
                isPrime = 0;    // = - присвояване (виж [Оператор за присвояване](../03_Оператори.md#1-оператор-за-присвояване)) (намерихме делител, значи не е просто)
                break;         // break - прекъсване на цикъла
            }                  // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
        }                      // } - край на съставния оператор (тяло на for) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    }                          // } - край на съставния оператор (тяло на if-else) (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    if (isPrime == 1) {        // if - условен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор)), (условие) - кръгли скоби, == - сравнение (виж [Оператори за сравнение](../03_Оператори.md#7-оператори-за-сравнение))
        printf("Просто число\n");  // printf - функция за изход от #include <stdio.h>
    } else {                   // else - алтернативен оператор (виж [Условен оператор](../03_Оператори.md#2-условен-оператор))
        printf("Не е просто число\n");  // printf - функция за изход от #include <stdio.h>
    }                          // } - край на съставния оператор (виж [Съставен оператор](../03_Оператори.md#3-съставен-оператор))
    
    return 0;
}

Пример: - 7 е просто (делители: 1, 7) - 8 не е просто (делители: 1, 2, 4, 8)

Обобщение на важните техники

  1. Въвеждане на n числа: Използваме for цикъл от 1 до n
  2. Работа с цифри: Използваме % 10 и / 10 в цикъл
  3. Проверка за четност: n % 2 == 0
  4. Проверка за делимост: n % k == 0
  5. Намиране на мин/макс: Инициализираме с първото число, след това сравняваме
  6. Броячи: Инициализация с 0, увеличаване с 1
  7. Суми: Инициализация с 0, добавяне на стойности
  8. Обръщане на число: reversed = reversed * 10 + digit
  9. Проверка за палиндром: Сравняваме оригиналното с обърнатото
  10. Прости числа: Проверяваме за делители между 2 и n-1
